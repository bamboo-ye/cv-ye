核心工作内容

1. 主导数据链路设计与实现

构建从数据采集到流处理的完整数据链路，实现端到端准实时处理。

  • 用 Golang 编写定向爬虫获取 2018-2022 年 CVPR 论文（7,391 篇）及作者数据（12,068 条）
  • 开发 Kafka 生产者客户端，实现爬虫与处理层解耦
  • 搭建 Flink 流处理引擎，消费 Kafka 数据后通过状态编程完成实时清洗
  • 实现 DOI 去重、作者姓名标准化等数据清洗逻辑
  • 用 Flink SQL 解析论文摘要提取关键词
  • 最终将结构化数据写入 MySQL，端到端数据处理延迟控制在 30 秒内

2. 负责后端服务与可视化支撑

构建高性能API服务与实时数据可视化展示。

  • 基于 Golang Gin 框架开发 RESTful API
  • 直接读取 Flink 处理后写入 MySQL 的结构化数据
  • 利用 Go Routine 并发特性优化查询性能
  • 监听 MySQL 数据变更
  • 使用 ECharts 以动态词云、折线图等呈现数据
  • 使用 WebSocket 同步推送数据更新
